Use WS2811/WS2812 LEDs with an AVR

Introduction

This project is about using an WS2811 or WS2812 lightstribe with an AVR controller. To handle up to 250 LEDs at the same time I chose an Atmega328p because of the RAM amount. If you want to handle less LEDs you can use most parts of this project with every AVR. The AVR is programmed to receive the configuration data over UART so you can use the LEDs with an ESP8266 or every other thing that provides such an interface. You can also use an FTDI to control the LEDs.
